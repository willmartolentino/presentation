@if

It allows us to write conditions in our Sass. This kind of feature is mostly useful when writing what we generally refer to as SassScript, writing reusable mixins and functions for Sass.

Example:

SASS

@mixin country_color($country) {

@if $country == philippines {

color: blue; }

@else if $country == spain {

color: yellow; }

@else {

color: red; }

}


.england { @include country_color(england); }

.philippines { @include country_color(philippines); }



CSS

.england { color: red; } // default color

.philippines { color: blue; }
