Stop Repeating Yourself with @each

What is @each?

It's a trick to keep your Sass DRY "Don't Repeat Yourself". It's a way of copying the same styling for a lot of different variables.

Example:

SASS

@each $lang in english, japanese, korean {

#{$lang}_language {

background-image: url("/image/#{$lang}.png");

}

}



CSS

.english_language {

background: url("image/english.png"); }

.japanese_language {

background: url("image/japanese.png"); }

.korean_language {

background: url("image/korean.png"); }
