Stop Repeating Yourself with @each

What is @each?

It's a trick to keep your Sass DRY "Don't Repeat Yourself". It's a way of copying the same styling for a lot of different variables.

Example:

SASS

@each $member in thom, jonny {

#{$member}_picture {

background-image: url("/image/#{$member}.png");

}

}



CSS

.thom_picture {

background: url("image/thom.png"); }

.jonny_picture {

background: url("image/jonny.png"); }
