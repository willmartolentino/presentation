SASS in the command line

What is Mixins?

So, mixins let you define common properties once, then re-use them.

Mixins are define by using @mixin directive, and if you want to use the mixin, you have to use the $include directive to include the mixin.

Example:

SASS

@mixin .myMixin {

font-size: 18px;

background-color: white;

}


.myStyle {

@include myMixin;

}



CSS

.myStyle {

font-size: 18px;

background-color: white;

}
