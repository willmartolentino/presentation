A Glimpse of SASS

CSS Extensions

Nested Rules - This helps avoid repetition of parent selectors

Example No. 1:

#main p {

color: #00ff00;

width: 97%;


.redbox {

background-color: #ff0000;

color: #000000;

}

}

is compiled to:

#main p {

color: #00ff00;

width: 97%;

}


#main p .redbox {

background-color: #ff0000;

color: #000000;

}
