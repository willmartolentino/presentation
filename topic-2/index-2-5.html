SASS Output Formatting

SASS provides several options to control how the .scss code is formatted when compiled into CSS

these are specified using the --style option in the terminal/command line

  • Nested - Default format, Indented
  • Expanded - most human-friendly readable forma t
  • Compact - CSS output is condensed, but still readable
  • Compressed - minimized output, suitable for production environments
