SASS in the command line

Basic commands you should know for compiling SASS files

  • sass -?
  • sass --update filename
  • sass --watch (source directory):(destination directory)
  • Command for HELP
    sass -?

  • Command to convert a .scss file to a .css file
    sass --update filename
    sass --update original_filename.scss:new_filename.css
    Command to convert a bunch of .scss files or a directory containing the .scss files
    sass --update (source directory):(destination directory)

  • Command to convert a .scss file/s to a .css file/s and once you use this command you can freely edit the file/s you want without typing the --update command in the terminal over and over again because it automatically converts the file/s when you run it. You can stop this command by pressing Ctrl-C.
    sass --watch (source directory):(destination directory)
