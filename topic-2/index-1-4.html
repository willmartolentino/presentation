Nested Properties

Example No. 1: Setting a bunch of Properties that have the same namespace

.funky {

font: {

family: fantasy;

size: 3.0em;

weight: bold;

}

}

is compiled to:

.funky {

font-family: fantasy;

font-size: 3.0em

font-weight: bold;

}
