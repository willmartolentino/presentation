Referencing Parent Selectors using "&"

Example No. 2: Supposing that you have an element which 'id' has a value of 'main-sidebar'

#main {

color: black;


&-sidebar { border: 1px solid; }

}

is compiled to:

#main { color: black; }


#main-sidebar { border: 1px solid; }
