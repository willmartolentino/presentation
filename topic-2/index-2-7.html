Interpolation

What is Interpolation?

Basically it means to "put this there".

It also means that the very name of a property or selector could be dynamically generated.

How to use Interpolation?

Just wrap the name of a variable in #{}.

Example:

SASS

@mixin car_make($car_make, $car_color) {

// Set the $car_make with "_make" at the end as a class

.car.#{$car_make}_make {

color: $car_color;

width: 100px;

.image { background: url("images/#{$car_make}/#{$car_color}.png"); }

}

}


@include car_make("volvo", "green");



CSS

.car.volvo_make {

color: "green";

width: 100px;

}

.car.volvo_make .image { background: url("images/volvo/green.png"); }
